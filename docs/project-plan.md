# FundFlow NXT

A full-stack nonprofit fund accounting POC inspired by FinancialEdge NXT.

## Modules
1. General Ledger
2. Treasury
3. Accounts Payable
4. Grants Management
5. Budgeting and Reports

## Tech Stack
- Frontend: Angular
- Backend: ASP.NET Core C#
- Database: SQL Server
- Cloud: Azure
- AI: Copilot and Claude API

## Problem This Solves
- Bulk transaction clearing takes hours in FE NXT
- GL reconciliation has no AI support yet
- No plain-English explanation of financial anomalies
